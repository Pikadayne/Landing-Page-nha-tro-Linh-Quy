# Nhà Trọ Linh Quy - Landing Page

🏠 Landing page chuyên nghiệp cho doanh nghiệp nhà trọ "Nhà Trọ Linh Quy" được xây dựng bằng **React + Tailwind CSS**.

## 📋 Nội Dung

- ✨ **Hero Section** - Trang chủ hấp dẫn với CTA rõ ràng
- 🏘️ **Giới Thiệu** - Giới thiệu về nhà trọ và ưu điểm
- 🛏️ **Loại Phòng** - Hiển thị 3 loại phòng với giá và tiện nghi
- ⚡ **Tiện Nghi** - Danh sách các tiện nghi có sẵn
- 🖼️ **Gallery** - Bộ sưu tập hình ảnh placeholder
- 📍 **Bản Đồ** - Bản đồ Google Maps vị trí nhà trọ
- ⭐ **Review** - Bình luận từ khách hàng
- 📞 **CTA Cuối** - Khuyến khích gọi điện/Zalo
- 👣 **Footer** - Thông tin liên hệ

## 🎨 Đặc Điểm

- ✅ Responsive design - hoạt động trên mọi thiết bị
- ✅ Sticky CTA button - luôn hiển thị khi scroll
- ✅ Smooth animations - hiệu ứng chuyển động mềm mại
- ✅ Navigation menu - thực đơn sticky với responsive mobile
- ✅ Placeholder images - không sử dụng ảnh thật
- ✅ Branding colors - xanh lam + trắng

## 🚀 Cài Đặt & Chạy

### 1. Cài đặt dependencies
```bash
npm install
```

### 2. Chạy development server
```bash
npm run dev
```

Server sẽ chạy tại: `http://localhost:3000`

### 3. Build cho production
```bash
npm run build
```

### 4. Preview production build
```bash
npm run preview
```

## 📱 Công Nghệ

- **React 18** - UI library
- **Vite** - Build tool cho development nhanh
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Icon library
- **PostCSS** - CSS processing
